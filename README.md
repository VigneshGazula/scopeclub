## ğŸ¯ About The Project

The **SCOPE Club Official Website** serves as the central digital hub for the **School Of Programming Excellence (SCOPE)** at MLR Institute of Technology. This platform is designed to bridge the gap between academic learning and real-world technical excellence by providing students with resources, event information, and a vibrant community space.

### Mission Statement

SCOPE Club is dedicated to instilling a strong coding culture through collaboration, innovation, and hands-on learning experiences across multiple technical domains including **AI/ML, Open Source Development, Game Development, Cloud Computing, and Full-Stack Web/App Development**.

### Project Goals

- **Centralized Information Hub**: Provide a single source of truth for all club activities, events, and announcements
- **Community Building**: Foster connections between students, alumni, and industry professionals
- **Showcase Excellence**: Highlight student projects, achievements, and technical innovations
- **Resource Accessibility**: Offer learning materials, workshop recordings, and technical documentation
- **Event Management**: Streamline registration and participation in hackathons, workshops, and competitions

## ğŸš€ Getting Started

Ensure you have the following installed:
- **Node.js** (v16.x or higher) â†’ [Download](https://nodejs.org/)
- **npm** or **yarn** â†’ Comes with Node.js
- **Git** â†’ [Download](https://git-scm.com/)

### **Installation Steps**

1. **Clone the repository**
   ```bash
   git clone https://github.com/vajjipartisatwikraj/scopeclub.git
   cd scopeclub
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Set up environment variables** (if applicable)
   ```bash
   cp .env.example .env.local
   # Edit .env.local with your API keys
   ```

4. **Start the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

5. **Open your browser**
   ```
   Navigate to http://localhost:3000
   ```

### **Build for Production**
```bash

npm run preview  # Test production build locally
```
